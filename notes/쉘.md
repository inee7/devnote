# 출력

간단하게 짧은 문서를 출력하려면 **cat**
긴 출력 결과를 끊어서 보고 싶으면 **more**
긴 문서를 조금 더 꼼꼼하게 확인하고 싶으면 **less**
문서의 머리부분을 확인하고 싶으면 **head**
꼬리부분을 보고 싶거나 로그를 실시간으로 확인하고 싶으면 **tail**
