# 7. 컬렉션
일급컬렉션으로 컬렉션과 관련된 로직을 캡슐화 하자. 
컬렉션 인스턴스변수와 컬렉션 변수에 잘못된 값이 할당되지 않게 막는 메소드가 있다. 
```kotlin
class Party {
	private final List<Member> members;

	void add(Member newMember) {
		List<Member> adding = new ArrayList<>(members);
		members.add(newMember);
		return new Party(adding); // 부수효과를 막기위해 새로운 객체 리턴 
	}

	List<Memeber> members() {
		return members.unmodifiableList(); // 외부에서 변경 못하게 
	} 
}
```
