# 리터럴함수란
리터럴 함수는 코드에서 함수를 값처럼 표현하는 방법 중 하나입니다. 이는 함수를 직접 작성하고 변수에 할당하거나 다른 함수에 전달하는 등의 유연한 사용을 가능케 합니다.

간단한 예를 통해 설명하겠습니다. Kotlin에서의 람다 리터럴 함수는 중괄호 `{}`로 감싸진 파라미터와 본문으로 이루어져 있습니다. 아래는 두 정수를 더하는 람다 리터럴 함수의 예시입니다:

```kotlin
val sum: (Int, Int) -> Int = { a, b -> a + b }
```

이 코드에서 `sum`은 `(Int, Int) -> Int` 타입의 변수로, 두 개의 정수를 받아 정수를 반환하는 함수를 나타냅니다. 람다 리터럴 함수는 중괄호 내의 코드 블록 `{ a, b -> a + b }`로 정의되어 있습니다.

리터럴 함수의 간단한 사용 예시를 보면, 다음과 같이 함수를 호출하거나 다른 함수에 전달할 수 있습니다:

```kotlin
val result = sum(3, 5) // result에는 3과 5를 더한 값(8)이 저장됨

fun operateOnNumbers(a: Int, b: Int, operation: (Int, Int) -> Int): Int {
    return operation(a, b)
}

val multiplicationResult = operateOnNumbers(2, 4, { x, y -> x * y }) // multiplicationResult에는 2와 4를 곱한 값(8)이 저장됨
```

이처럼 리터럴 함수는 함수를 값으로 취급할 수 있게 해주어 코드를 간결하게 만들고, 고차 함수와 함께 사용될 때 더욱 강력한 표현력을 제공합니다.
