# 어댑터 패턴

## 한 줄 요약

어떤 인터페이스를 다른 인터페이스로 변환하여 호환성을 제공하는 패턴

## 핵심 정리

- 어떤 인터페이스에서 다른 인터페이스로 변환하고자 할 때 사용
- 확장 함수를 통해 다른 클래스 타입으로 변경 가능
- 일반적인 어댑터 메소드 예: `stream.toList()`

## 상세 내용

### 코틀린에서의 구현

확장 함수를 통해 또 다른 클래스 타입으로 변경한다.

```kotlin
fun UsbMini.toUsbTypeC(): UsbTypeC {
    val hasPower = this.hasPower == Power.TRUE
    return object : UsbTypeC {
        override val hasPower = hasPower
    }
}
```

### 실무 활용

코틀린 표준 라이브러리에서 제공하는 많은 변환 함수들이 어댑터 패턴의 예시다:
- `stream.toList()`
- `list.toSet()`
- 기타 `to*()` 형태의 변환 함수들
