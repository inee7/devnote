# 4. 불변
함수에서 부수효과 발생하기 좋다. 
이유는 가변 인스턴스 사용, 매개변수 변경, 전역변수 변경, 외부 데이타 조작 (파일,I/O) 등이다.
부수효과를 막으려면?
상태를 매개변수로 받고 상태를 변경하지 않고 값을 리턴하기만 하는 함수가 이상적이다.
그러면 함수에서 인스턴스 변수를 사용하지 말라는건가?
아니다. 인스턴스변수는 불변으로 만들어 영향이 전달되지 않게 할 수 있다, 
그리고 객체지향언어는 클래스내부는 동일 인스턴스 변수를 사용하는것을 허용한다. 

불변이라면 값을 변경할때 인스턴스를 새로 생성해야한다.
