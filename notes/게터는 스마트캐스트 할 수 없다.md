# 게터는 스마트캐스트 할 수 없다 
```kotlin
val name: String? = "aaa"
val surname: String = "bbb"
val fullName: String?
    get() = name?.let { "$it $surname" }

val fullName2: String? = name?.let { "$it $surname" }

fun main() {
    if (fullName != null) {
        println(fullName.length) // ERROR
    }

    if (fullName2 != null) {
        println(fullName2.length)
    }
}

```
