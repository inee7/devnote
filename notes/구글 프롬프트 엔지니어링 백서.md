# 📘 구글 프롬프트 엔지니어링 백서 요약

---

## 1. 프롬프트 엔지니어링의 중요성

프롬프트는 단순한 질문이 아닙니다. 모델이 어떤 결과를 출력할지 결정짓는 가장 중요한 입력입니다.

- 모델 종류
- 학습 데이터
- 문장 구조, 문체
- 설정값 (Temperature, Top-K 등)

이 요소들이 종합적으로 작용하기에 프롬프트는 전략적으로 설계되어야 합니다.

---

## 2. 출력 품질을 제어하는 주요 설정

| 설정 항목 | 설명 |
| --- | --- |
| **Temperature** | 창의성 제어 (0: 결정적, 1: 창의적) |
| **Top-K / Top-P** | 예측 후보 제한 방식 (개수 vs 누적 확률) |
| **Output Length** | 생성될 최대 토큰 수 지정 |

### Output Length란?

- 생성될 텍스트의 최대 길이 (단위: 토큰)
- 너무 짧으면 문장이 중간에 끊기고, 너무 길면 요점이 흐려질 수 있음

**활용 예시**:

- 요약: 100~200 토큰
- 에세이: 500~1000 토큰
- 질문 응답: 50~150 토큰

### Top-K란?

- 모델이 다음 단어를 예측할 때 **확률이 높은 상위 K개 후보 단어만 남기고 나머지는 제거**
- 예: K=3이면 상위 3개 단어 중 하나만 선택됨
- 장점: **정확하고 예측 가능한 결과 유도**

### Top-P란? (Nucleus Sampling)

- 모델이 다음 단어를 예측할 때, **누적 확률이 P(예: 0.9)를 넘지 않는 후보들만 남김**
- 확률이 높은 단어부터 더해가며 누적합이 P를 넘는 순간 멈춤
- 예: P=0.9이면 상위 후보들의 확률 합이 90%가 넘을 때까지 포함
- 장점: **더 다양한 응답을 생성할 수 있어 창의적 작업에 유리**

---

## 3. 프롬프트 유형별 전략

- **시스템 프롬프트**: 모델의 기본 역할 설정 (예: "당신은 여행 가이드입니다.")
- **컨텍스트 프롬프트**: 배경 정보 제공 (예: 과거 대화, 외부 정보 등)
- **롤 프롬프트**: 특정 인물, 역할로 행동 유도 (예: "심리학자처럼 답변해 주세요")

---

## 4. 다양한 프롬프팅 기법 소개

- **제로샷**: 예시 없이 질문
- **원샷**: 예시 1개 포함
- **퓨샷**: 예시 3~5개 포함 (권장)
- **Chain of Thought (COT)**: 문제 해결을 단계별로 유도
- **Step-back Prompting**: 답변 생성 전 필요한 정보를 먼저 추출
- **Self-Consistency**: 같은 질문을 여러 번 해서 다수결로 결정

---

## 5. 최신 프롬프트 기법

### ✅ Tree of Thought

- 여러 사고 경로를 동시에 탐색하고, 가장 최적의 응답을 선택하는 구조적 사고 흐름

### ✅ ReAct (Reasoning + Acting)

- LLM이 "생각 → 행동 → 피드백 → 반복" 구조로 문제를 해결
- 외부 도구 호출, 검색, 계산 등이 가능함

### ✅ AP (Automatic Prompt)

- 다양한 표현 방식으로 프롬프트를 LLM이 스스로 생성
- 챗봇 훈련, 검색 다양성 확보에 유리함

---

## 6. 프롬프트 설계 실전 팁

- **명령형 문장 사용**: "Generate", "Act as" 등
- **Do 중심 표현**: 하지 말라는 것보다 해야 할 것을 강조
- **형식 명시**: JSON, 리스트, 단락 수 등 명확히
- **문장 단순화**: 짧고 명확한 표현이 가장 효과적
- **추천 동사**: analyze, classify, compare, summarize

## 7. 프롬프트 관리와 실험

| 항목 | 예시 |
| --- | --- |
| 이름 | 피자 주문 JSON |
| 목적 | 주문 자동화 |
| 모델 | GPT-4 |
| 설정값 | Temp 0.2 / Top-P 0.95 |
| 예시 | 다양한 주문 문장 |
| 결과 평가 | 성공 / 실패 / 개선 필요 |

프롬프트는 여러 형태로 시도하고, 결과를 비교하며 관리해야 발전합니다.

프롬프트 기록을 통해 최적화된 스타일을 찾고 재활용할 수 있습니다.
